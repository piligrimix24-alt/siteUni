Сайт "Сад на подоконнике"

Что это:
База знаний о съедобных комнатных растениях. Можно выращивать мяту, базилик, укроп, петрушку, салаты, лук, микрозелень, помидоры, перцы и редис на подоконнике.

Для кого:
Студенты и молодые люди, которые хотят свежую зелень дома, но боятся, что растения погибнут. Сайт помогает выбрать растение, понять, как за ним ухаживать, и что из него приготовить.

Что есть на сайте:
- Статьи о растениях (уход, сорта, проблемы)
- Гайды по уходу для разных категорий
- Рецепты из выращенной зелени
- AI-ассистент, который отвечает на вопросы по базе знаний
- Тёмная и светлая тема

Технологии:
- VitePress — генератор статического сайта
- Markdown — контент
- Python + FastAPI — сервер для AI-ассистента
- ChromaDB — векторная база знаний
- OpenRouter — бесплатный доступ к LLM
- GitHub Pages — хостинг

Как запустить локально:
1. Установите Node.js и Python
2. Клонируйте репозиторий
3. В папке siteUni выполните: npm install и npm run docs:dev
4. В папке backend: создайте виртуальное окружение, установите зависимости, запустите python openrouter_server.py
5. Сайт откроется на localhost:5173

Для AI-ассистента нужен API-ключ OpenRouter.
