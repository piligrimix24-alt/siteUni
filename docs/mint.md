# Мята

Мята — ароматное растение, которое легко вырастить на подоконнике. Её листья можно добавлять в чай, коктейли, салаты и десерты.

## Уход
- **Свет:** Яркий рассеянный.
- **Полив:** Регулярный, земля не должна пересыхать.
- **Температура:** 18-22°C.

## Как использовать в еде
- Чай со свежей мятой
- Мохито
- Мятный соус к ягнёнку
- Украшение десертов

## Размножение
Черенками — поставьте веточку в воду, через неделю появятся корни.
